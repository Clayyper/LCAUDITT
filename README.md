# Comparador CLT x PJ — versão visual

Projeto em Node.js + Express com interface moderna e interativa.

## Como rodar

```bash
npm install
npm start
```

Abra no navegador:

```bash
http://localhost:3000
```

## Melhorias desta versão
- visual mais moderno
- cartões com destaque
- barras comparativas
- indicadores de vantagem
- inputs com atualização em tempo real
- layout responsivo

## Observação
As fórmulas continuam aproximadas com base no print enviado e podem ser refinadas depois.
